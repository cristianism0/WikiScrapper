# Wikipedia Curiosities Scrapper

This project is used to scrap the `Did you know` section and generate a `.csv` file with the content scrapped.

## Requirements
- Git. *for clone this repo*
- [BeautifulSoup](https://beautiful-soup-4.readthedocs.io/en/latest/)
- [Requests](https://pypi.org/project/requests/).

## How to use
1. Clone the repository:
```bash
git clone https://github.com/cristianism0/WikiScrapper
```
2. Go to `main.py` and paste the wikipedia URL link with Curiosities.
    * If you don't know where to find the page, search for: `Wikipedia: Did you know`
  
3. Run `main.py`
```bash
python main.py
```
